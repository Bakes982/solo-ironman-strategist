# Setting Up IntelliJ IDEA for RuneLite Plugin Development

1. **Install IntelliJ IDEA** (Community or Ultimate).
2. **Install JDK 11** (AdoptOpenJDK recommended).
3. **Fork the RuneLite repository** on GitHub.
4. **Clone your fork locally** and open in IntelliJ.
5. **Import as Gradle project** (if prompted).
6. **Set up run configurations** for plugin development.
7. **Develop your plugin in `plugin/modern_meta_task_overlay/`**.

See the official RuneLite plugin guide for more details.
